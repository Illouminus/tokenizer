Ebaillot42 (EB42) Whitepaper
This token is created as part of the School 42 Tokenizer project.
Purpose: EB42 is a test token designed to demonstrate the core functionalities of the BEP-20 standard on the BNB Smart Chain.
Features:

Transfers: Users can send and receive EB42 tokens seamlessly.

Burnable: Token holders can destroy their own tokens to reduce the total supply.

Pausable: The contract owner holds the privilege to pause all token transfers in case of an emergency or a security breach, protecting the community.
Usage: To use this token, add the contract address to your Web3 wallet (e.g., MetaMask) on the BNB Smart Chain Testnet. You will need a small amount of tBNB for gas fees to execute transactions.
